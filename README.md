# Desafio Frontend - Flugo

Este projeto é um formulário de cadastro multi-step desenvolvido para o desafio técnico da vaga de Desenvolvedor Frontend na Flugo. 

A aplicação coleta os dados do funcionário em etapas, realiza validações e salva as informações em tempo real no banco de dados.

## Tecnologias Utilizadas
- ReactJS com TypeScript
- Material UI (MUI)
- Vite
- Firebase (Firestore)

## Link do Projeto
- **Aplicação Online (Vercel):** https://desafio-firebase-vander.vercel.app

## Como rodar o projeto localmente

Siga os passos abaixo para testar o projeto na sua máquina.

### Pré-requisitos
- Node.js instalado
- Git instalado

### Passo a passo

1. Clone este repositório:
git clone https://github.com/ovandersongabriel-hue/desafio-firebase-vander.git

2. Acesse a pasta do projeto:
cd desafio-firebase-vander

3. Instale as dependências:
npm install

4. Inicie o servidor local:
npm run dev

5. O terminal exibirá um link local (geralmente http://localhost:5173). Clique nele ou cole no seu navegador para ver a aplicação rodando.
